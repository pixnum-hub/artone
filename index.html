📖 Artone User Guide

5 Steps to Start Drawing

1️⃣
Pick a tool from the left sidebar. Start with 🖊 Pen for freehand drawing.
2️⃣
Choose a colour from the header 🎨 Stroke picker or click any palette swatch.
3️⃣
Set Size & Opacity with the sliders in the header.
4️⃣
Draw on canvas — click/drag on desktop, touch/drag on mobile.
5️⃣
Save using ⬇ PNG or ⬇ JPG in the header quick-bar or right sidebar.

Interface Layout

◀
Left sidebar — all drawing tools and shape tools, scrollable.
▶
Right sidebar — canvas ops (flip, rotate, resize), 9 filters, and export buttons.
▲
Header — theme, stroke/fill colours, palette, size, opacity, stroke style, blend mode, quick actions.
▲
Centre top bar — Sym, Grid, Ruler, Mini-map, Layers panel, Zoom, Background, Import.
📚
Layers panel — slides out from the right. Manage up to 10 layers with visibility, opacity, merge, duplicate, and flatten.

Freehand Tools

🖊
Pen — smooth anti-aliased line. Respects Blend Mode and Dash style.
🖌
Brush ▾ — 30 brush styles via the dropdown picker.
💨
Spray — airbrush scatter. Hold still to build density.
✏
Pencil — thin grainy freehand sketching tool.
◻
Erase — clears to transparent on current layer only.
👆
Smudge — blends nearby pixels in stroke direction.
🔁
Clone — Alt+click to set source, then paint to clone that area.

Special Tools

🪣
Fill — flood-fills a bounded area.
🔬
Pick — eyedropper: click any pixel to sample its colour.
T
Text — click canvas to place text. Choose font, style, and size in the popup.
✂
Crop — drag to select, releases to crop canvas to that region.

Shapes

╱▭■▢
Line, Rect, FRect, RRect — line, stroke rect, filled rect, rounded rect. Hold Shift on Line for 45° snap.
○●△▲
Ellipse, FEllipse, Tri, FTri — stroke and filled versions.
★⬡⬠➜
Star, Hex, Pentagon, Arrow — polygon and arrow shapes.

Using Brushes

▸
Click 🖌 Brush ▾ in the left sidebar to open the picker showing all 30 brushes in a scrollable grid. Click any brush to activate it.
▸
Size and Opacity sliders apply to all brushes. Try low opacity (20–50%) for Watercolor, Chalk, and Crayon.

Layer System

📚
Click Layers in the centre top bar to open the Layers panel on the right edge.
＋
Add — creates a new transparent layer above the current one. Up to 10 layers.
🗑
Delete — removes the selected layer (minimum 1 layer is preserved).
⬇
Merge Down — flattens the current layer onto the one below it.
⎘
Duplicate — copies the current layer above itself.
⊕
Flatten — merges all layers into one (undo-safe).
👁
Click the eye icon on any layer to toggle visibility.
◻
The layer Opacity slider in the panel controls the composite opacity of the entire layer.

9 Filters (right sidebar)

◑
Grayscale — luminosity-weighted desaturation.
⬡
Invert — inverts all pixel colours.
🟫
Sepia — warm vintage tone using standard sepia matrix.
💧
Blur — 3px Gaussian soft blur via CSS filter.
🔆
Sharpen — convolution kernel sharpening.
☀
Brightness/Contrast — sliders from -100 to +100. Undo-safe.
🎭
Posterize — reduces colour levels to 4 for a graphic art effect.
⬛
Pixelate — applies mosaic pixelation (16px cells).
🔲
Vignette — adds a dark radial gradient vignette frame.

Export Options

⬇
PNG — lossless with transparency. All layers + background are merged in the export.
⬇
JPG — compressed, white background, smaller file size.
⬇
WebP — modern format, smallest file size with good quality.
⎘
Copy — copies merged canvas as PNG to system clipboard.
